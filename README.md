#  Shitty Painter
Welcome to Shitty Painter! This program is a lightweight and fun painting tool designed for those who just want to doodle or create quick sketches without the need for complex features. It's a simple painting tool that was made in 2 days or so. This was just made for fun and I do not plan on making frequent updates to it.

##  Features
- Import/Export
- Pencil
- Eraser
- Color Picker
- Pixelator
- Fish

##  How To Install
1. Download the zip file from releases
2. Open the zip file
3. Export the folder
4. Open the exe
